Este trabajo consiste en un blog, este blog está dedicado para poder subir tareas del segundo bimestre de Sexto. Fue creado con mucha dedicación. Se utilizó para diseño el Framework tailwindcss, un framework usado para la personalización de páginas web. Aquí se subirán todos los trabajos del presente bimestre. Este trabajo sirvió para poder trabajar en una página web profesional y personal, además de eso se aprendió como subir una página web de manera gratuita a internet. Se trabajó la página web con total libertad y con el diseño que el alumno tenía pensado. Y aquí está mi idea de blog personal.
